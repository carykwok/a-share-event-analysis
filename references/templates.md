# A股热点事件解析 · 20 模板库（合规版）

每个模板含：触发场景 / 主副标题示例 / 5 维度 / 字数 / Prompt。

**全局合规红线**（每个模板 Prompt 内都已嵌入，生成稿件时严格遵守）：
- 不写股票代码，不列具体公司+代码组合；最细颗粒度到"某细分环节龙头"/"某产业链环节"
- 不做行情绝对性判断（不用"必然/一定/见底/反转/突破"）
- 不给投资操作建议（不用"买入/建仓/超配/低配/规避/建议配置"）
- **不做板块估值区间判断**：因缺乏可靠权威数据源，禁止在全文（尤其风险提示）使用"板块整体 PE 高位 / 估值偏高 / 估值泡沫 / 处于偏高区间 / 估值消化压力"等断言。如须提及估值相关风险，仅可采用"估值波动风险"、"业绩证伪带来的股价回调风险"等不涉及具体区间的中性表述
- 使用"可能/或/有望/预计/或将/料/大概率"等缓和措辞
- 结尾附固定免责声明

## 1. 货币财政政策

### 1A 货币政策宽松
- **触发**：央行降息/降准、MLF/LPR 利率下调
- **主标题示例**：「降息落地，宽松周期开启」
- **副标题示例**：货币政策转向如何影响A股估值体系？
- **字数**：450–550 字
- **5 维度**：① 政策力度 vs 预期差 ② 利率下行→折现率→估值弹性传导 ③ 可能受关注板块：地产/消费/成长方向 ④ 流动性溢出与风险偏好观察 ⑤ 历史宽松周期回溯
- **Prompt**：
```
你是专业A股市场分析师，请根据以下信息撰写行情解析文章。

核心事件：{例：央行宣布全面降准0.5个百分点，释放长期资金约1万亿元，同时下调7天逆回购利率10bp}
市场反应：{当日主要指数及板块涨跌}
撰写时间：{日期}

合规红线：
- 不写股票代码，不列具体公司名+代码
- 不做绝对性走势判断，使用"可能/或/有望/预计"等缓和措辞
- 不给操作建议，改为"相关行业板块可关注"
- 结尾附免责声明

输出格式：
（1）主标题：≤15字，点明政策方向，客观描述事件
（2）副标题：20-30字，提出分析视角或关键问题
（3）正文约500字：
   政策解读（100字）：核心条款 + 与市场预期的差异点
   传导逻辑（100字）：降息→资金成本→折现率→估值弹性的链条描述
   板块观察（150字）：3-5个可关注的行业方向，每个附2句逻辑说明（仅到行业/链条颗粒度）
   历史参照（80字）：历史上类似政策后市场的表现规律
   前瞻观察（70字）：后续政策空间的观察窗口（不做走势断言）+ 风险提示
（4）语言简练专业，每个观点有逻辑支撑
（5）结尾附："本文仅为信息梳理与行业观察，不构成任何投资建议，市场有风险，决策需谨慎。"
```
