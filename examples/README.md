# Examples · 产出示范

本目录收录使用 skill 实战跑过的对客合规稿件样例，用于：
- 新用户快速对齐产出颗粒度与字数
- 参考各大类模板的实际写法
- 合规红线的具体落地示范

**时效性说明**：以下样例对应 2026 年 4 月的事件与数据点，仅作写作框架参考。所引用的数据、行情、产业链信息均来自发稿时点的联网公开信息源，实际使用时请以当时最新数据为准。

**合规声明**：所有样例遵循 SKILL.md 中的四条合规红线——不写股票代码 / 不做走势绝对判断 / 不给投资操作建议 / 不做板块估值区间判断。

## 样例共性结构

每篇样例均包含：
- **主标题**（≤15 字）
- **副标题**（20–35 字）
- **正文**（按模板规定字数，拆分为若干段，每段对应一个维度）
- **风险提示**（5 项客观因素，不含估值区间断言）
- **免责声明**（固定文本）
- **Sources**（一/二级权威信息源的 markdown 链接）
