**产业链分层**：A 股相关产业链呈现典型三段式结构。上游：原材料与核心零部件供应环节，业绩弹性最大。中游：整机制造与封装测试环节，毛利水平稳定。下游：品牌销售与应用服务环节，规模效应领先。

**时序节奏**：产业周期关键节点梳理如下，2026年1月：行业迎来补库存拐点；2026年2月：龙头企业公布扩产计划；2026年3月：下游订单环比明显回暖；2026年4月：板块业绩预告密集披露。

**三段展望**：按照时间维度拆解节奏。短期：业绩预告驱动板块情绪修复。中期：产能释放节奏决定毛利持续性。长期：技术迭代与国产替代双轮驱动。

**节奏判断**：本轮周期传导路径可用 "政策 → 需求 → 产能 → 业绩" 链条描述，顺序清晰、节奏分明。

**风险提示**：①供给侧产能释放快于预期，可能压制价格弹性；②下游库存周期波动影响需求节奏；③海外订单受地缘因素扰动；④原材料价格上涨挤压毛利空间。
